## 🧩 Features

- ✅ Modular Header and Footer layout  
- 🖼️ Story Grid / Gallery view  
- 📝 Submit Form with Validations  
- 🔍 SEO & Metadata  
- 🎯 Banner / Hero Section  
- 📱 Fully Responsive Design  
- 🌐 React Router Integration  
- ⚡ Lightning-fast Builds with Vite  
- 🎨 Styled with Tailwind CSS
